title: Hack Assembler
description:  A C++ implementation of an assembler for the 16-bit Hack computer architecture, based on the specifications from "The Elements of Computing Systems" (Nand2Tetris). This program translates assembly code written in the Hack language into binary machine code that can be executed by the Hack computer. It processes both A-instructions (addressing) and C-instructions (computation), handling symbolic labels and variables according to the Hack specification.
