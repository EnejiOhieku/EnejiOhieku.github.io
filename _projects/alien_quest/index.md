title: Alien Quest 2d Game
description:  Alien Quest is a 2D space shooter game built with Python and Kivy. This project was created immediately after graduating high school as a personal challenge to build a game engine from the ground up. Instead of relying on pre-built physics or high-level game engine features, I implemented the core physics, collision detection, and rendering logic myself to deeply understand the fundamentals of game development and advanced OOP.

skills: 
  - Trigonometry & Coordinate Geometry
  - Graphics Rendering
  - Object Oriented Progamming
  - Python
  - Kivy
