title: Connect N a simple strategy board game
description:  Connect N is a customizable strategy board game developed using Python and the Pygame library. Created as an high school senior, this application generalizes the mechanics of classic games like Tic-Tac-Toe and Connect Four into a scalable, grid-based experience. The game allows players to compete on a square grid of variable size (NxN), supporting both local multiplayer and single-player modes against a Heuristic-based algorithm.
